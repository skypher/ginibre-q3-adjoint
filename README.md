# Full adjoint-generated Ginibre Q3

This is the official publication repository for Leslie P. Polzer's proof of
Ginibre's condition Q3 on the closed multiplicative convex cone generated by
the real adjoint character of a compact connected Lie group with simple Lie
algebra.

The repository contains only the Ginibre-Q3 manuscripts, exact and
directed-rounding certificate programs, accepted certificate data, primary
source audits, replay records, and referee-side verification tools.  It does
not contain sources or research notes from any other project.

The complete publication package is in [`ginibre_q3/`](ginibre_q3):

- `paper_full.pdf` contains the load-bearing Parts I--II;
- `full_q3_extension.pdf` contains Part III and the full signed-product
  hierarchy;
- `paper.tex`, `paper_full.tex`, and `full_q3_extension.tex` are the manuscript
  sources; and
- `REPLAY.md` and `FULL_Q3_DISTRIBUTED_REPLAY.md` define the reproducibility
  contract.

From the repository root, the principal commands are:

```sh
make -C ginibre_q3 clean-room-replay
make -C ginibre_q3 full-q3-extension
```

The first command is the complete Parts I--II clean replay and is
computationally expensive.  The second audits and rebuilds Part III.  Hardware,
software, arithmetic, and accepted-transcript details are documented inside
the publication directory.

## Scope

The affirmative theorem concerns the adjoint-generated cone.  It is not a
claim for all positive-definite functions on a nonabelian compact group.  The
paper gives an exact SO(3) value of -8/279 disproving that literal extension.
The intermediate cone of all real continuous central positive-definite
functions is not decided.
